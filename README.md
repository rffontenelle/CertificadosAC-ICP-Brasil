# CertificadosAC-ICP-Brasil
Store Brazilian Government's Certificate Authorities certificate chain

This repository was created to give me insight of what date the zip file ACcompactado.zip was changed and what file was added/removed/changed.

This helps maintaining the AUR package [ca-certificates-icp_br](https://aur.archlinux.org/packages/ca-certificates-icp_br).

Links:
- Repositório AC-Raiz (download root certificates, CRLs - certificate revogation lists, and all AC certificates archive): <br>https://www.gov.br/iti/pt-br/assuntos/repositorio/repositorio-ac-raiz
- AC certificates archive's SHA512 hash file: <br>https://acraiz.icpbrasil.gov.br/credenciadas/CertificadosAC-ICP-Brasil/hashsha512.txt
- Instituto Nacional de Tecnologia da Informação's hompage: <br>https://www.gov.br/iti/pt-br
